Build-inのwindow変換

公式Guideで紹介されているBeamのbuilt-inのwindow変換を紹介します。

Sliding time windowを見ると分かりますが、要素は少なくとも一つのwindowに属し、かつ複数個のwindowに属することもあり得ます。

  1. Fixed time windows
    固定幅で重なりを持たない window
  2. Sliding time windows
    固定幅で重なりを持たせた window
  3. Session windows
    到着時間が近いデータをまとめる window
  4. Global window
    全体で共通の window
